# Simple Web Quiz Application

This project provides the foundational structure for a simple, interactive web quiz. It leverages standard web technologies to present a user interface for multiple-choice questions, along with an architectural blueprint for expanding into additional modules like a calculator. ## Getting Started

Follow these steps to set up and run the project locally on your machine.

### Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/SireTallest/ALX_Simple_Quiz.git
    ```
2.  Navigate into the project directory:
    ```bash
    cd ALX_Simple_Quiz
    ```

### Usage

To view the static quiz interface, simply open the `index.html` file in your preferred web browser. While the core structure for the quiz and a calculator module are in place, the JavaScript logic for dynamic interactions and the CSS styling are intended for future implementation.

To view the main quiz page:
1.  Locate `index.html` in the root of the cloned directory.
2.  Right-click on `index.html` and choose "Open with" -> "Your Web Browser" (e.g., Chrome, Firefox, Edge).

## Features

*   **Quiz Interface Structure**: Provides a basic HTML layout for displaying quiz questions and multiple-choice answers.
*   **User Input Elements**: Includes radio buttons for answer selection and a submit button.
*   **Feedback Placeholder**: An area designed to display immediate feedback to the user after an answer submission.
*   **Modular Design**: Structured to easily integrate JavaScript logic for quiz functionality and a separate CSS file for styling.
*   **Extensibility**: Features a dedicated `calculator` directory, indicating readiness for a future standalone calculator module.

## Technologies Used

*   **HTML5**: For structuring the web pages and content.
*   **CSS3**: Planned for styling the visual presentation of the quiz and calculator (currently empty).
*   **JavaScript**: Planned for implementing interactive quiz logic and calculator functionality (currently empty).
